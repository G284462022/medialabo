今回はHTMLについて学んだ。さまざまな大きさの見出しの作り方や、リスト、説明や、URLの付け方、強調などを学んだ。また、今回教わったもの以外にもかなりの数があるため、それの調べかたを学んだ。これらの要素以外に、<br>などの空要素を学んだ。

第2回 CSS による Web ページの装飾

今回はCSSについて学んだ。文字のフォント、URLなどの色や、デザイン、などを設定した。他にも、影や、グラデーションをつけた。また、下線を引いたり、グリッドを使ったりした。そして、画像を文字の右上に移動させるなどの設定をした。

第3回 JavaScript の基本

オブジェクトやメンバー、javaScriptにおけるfor文について学び、演習をおこなった。オブジェクトは{..}でメンバーを指定し、データを項目ごとにまとめる。また、[]を用いて配列を作り、データを使用しやすくする。for文は、今まで習ったjavaとは違い、for(let n of 〜〜)のように記述する。これと配列を組み合わせ、プロブラムを書く。

数当てゲームについて

kazuate.htmlでは、javaScriptを用いて乱数と予想した数字を比較し、結果を表示した。ここでは、オブジェクトとメンバーを活用し判定結果を作り、javaで学んだif文を用いて判定した。あらかじめ判定後に表示する文面を記述し、if文の結果によって、それに合わせて出力をした。
